# Changelog

All notable changes to ecodyn are documented here.

## 1.0.3

- Added a test suite covering logistic growth, the logistic growth rate, and
  the Lotka-Volterra integrator (equilibrium, conserved invariant, trajectory
  shape).

## 1.0.2

- Documentation tidy-up and a runnable logistic example in the README.

## 1.0.1

- Cast the Lotka-Volterra initial conditions to float so integer inputs work.

## 1.0.0

- Initial release: `logistic`, `logistic_rate`, and `lotka_volterra` (RK4).
